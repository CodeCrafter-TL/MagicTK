    3.1 使用内置图形类创建基本图形#

    约 1271 个字 29 行代码 预计阅读时间 7 分钟

    3.1.1 创建一个矩形#

    矩形等图形是在 graphics.py 中的类。你可以在开发中直接使用 mt.xxx 调用它。

    创建矩形时，直接给一个变量赋值即可： 

    rect = mt.Rectangle(*args)
    当我们创建它时，需要传入一系列的参数。（这里指上面的 “*args” ）

    它们在源码中的定义为： 

    def __init__(self, master: "base.Frame", position: tuple[int | float, int | float],
                size: tuple[int | float, int | float], bg: str) -> None:
    第一个参数（省略 “self” ）是矩形的父容器。

    你只需要传入你创建的容器即可。例如： 

    frame = mt.Frame(root, *args)
rect = mt.Rectangle(frame, *args)
    第二个参数是矩形的位置，和前面一样使用列表或元组的形式传入。

    第三个是大小，也用列表或元组的形式传入。

    如果我们想让矩形在 x 为 20 ， y 为 20 的地方，大小为 80x30 ： 

    rect = mt.Rectangle(frame, [20, 20], [80, 30], *args)
    第四个参数是矩形的颜色，它可以设置为 Hex 16 进制颜色。

    如我们想让矩形的颜色为 “#ff0000” 红色（ RGB 为 255, 0, 0 ）： 

    rect = mt.Rectangle(frame, [20, 20], [80, 30], "#ff0000")
    这样你就可以得到一个颜色为红色的矩形。

    希望通过这一阶段，你已经可以创建一个矩形在你的容器上了。

    3.1.4 创建一个文本#

    文本是在 graphics.py 中的 Text 类。

    创建文本时，直接给一个变量赋值即可： 

    label = mt.Text(*args)
    当我们创建它时，需要传入一系列的参数。（这里指上面的 “*args” ）

    它们在源码中的定义为： 

    def __init__(self, master: "base.Frame", position: tuple[int | float, int | float],
                text: str, fg: str, *, placemode: typing.Literal['normal', 'usable'] = 'usable') -> None:
    第一个参数（省略 “self” ）是圆形的父容器。

    你只需要传入你创建的容器即可。例如： 

    frame = mt.Frame(root, *args)
label = mt.Text(frame, *args)
    第二个参数是文本的位置，和前面一样使用列表或元组的形式传入。

    第三个参数是文本的内容。

    如我们想让文本显示 “Hello MagicTK” ： 

    label = mt.Text(frame, [20, 20], "Hello MagicTK", *args)
    第四个参数是文本颜色，它可以设置为 Hex 16 进制颜色。

    如我们想让文本的颜色为 “#ff0000” 红色（ RGB 为 255, 0, 0 ）： 

    label = mt.Text(frame, [20, 20], 4, "#ff0000")
    这样你就可以得到一个颜色为红色的文本。

    第五个自选参数是文本的放置方式，有两个选择： normal 和 usable 。

    normal 表示文本的放置方式与 tkinter.Canvas.create_text() 一样，以文本中央的坐标来放置。

    usable 表示文本以左上角的坐标来放置。

    这个参数可以根据个人需要来进行决定。

    提示

    图形类提供了一些方法来进行后期调整，如 _position _size 等。它们和 Window 的方法相同，可以传入参数来改变属性。

    注意

    需要注意的是，如果参数为空不会返回值，会报错！

    希望通过这一阶段，你已经可以创建一个文本在你的容器上了。
